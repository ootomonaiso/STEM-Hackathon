# GitHub・Git 基礎

ハッカソンで必要になる最低限の Git 操作をまとめます。

## まず覚える用語
- リポジトリ: コードや資料をまとめる場所
- ブランチ: 作業を分けて進めるための枝
- コミット: 変更の記録

## 最小コマンド
```bash
# 変更状況の確認
git status

# 変更をステージに追加
git add .

# 変更を記録
git commit -m "メッセージ"
```
